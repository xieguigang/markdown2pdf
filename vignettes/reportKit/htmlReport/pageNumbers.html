pageNumbers {htmlReport} R Documentation

assign the page numbers to the html templates

Description

Usage

pageNumbers(report, orders,
    pageStart = 1);

Arguments

report

the html report template model. [as HTMLReport]

orders

the file basename of the html files to sort of the pages inside the report model object. [as string]

pageStart

page number count start, default is count start from 1. [as integer]

Details

the placeholder of the page number inside the template document text should be: 1. [#page] for the page number of current page 2. [#total_pages] for the total page numbers which is count from the template files.

Authors

reportKit

Value

this function returns data object of type HTMLReport.
